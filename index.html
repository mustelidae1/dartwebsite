About

Overview

Dart is an object-oriented/class-defined programming language created by Google. It’s most commonly used in development for multi-platform applications, including mobile, desktop, backend and web-apps. Its syntax and functionality is heavily influenced compared to other popular programming languages, like C, C#, and JavaScript.

History

Dart first appeared on October 10th, 2011 at the GOTO conference in Aurhus, Denmark by founders/developers Lars Bak and Kaspr Lund and was later published/maintained by Google. Since then, Dart has seen frequent periodic updates for its publically available SDK and compilers. Some noteworthy releases include Google’s Dart Editor (November 2011), an official Eclipse plugin (August 2012), a Dart to JavaScript (dart2js) compiler (March 2013), and an official JetBrains IDE plugin (April 2015).

Specifications

Dart is a multiparadigm object-oriented programming language with several aspects that make it ideal for multi-platform development. It is a statically typed, imperative, reflective, functional, and garbage-collected language that supports many helpful features, such as interfaces, mixins, and abstract classes/generics.

One of Dart’s most advertised and noteworthy features is the way it handles compilation and/or interpretation. Originally, the language only supported ARM & x64 machine code compilation for mobile, desktop, and backend applications. However, the dart2js compiler allows developers to compile their Dart programs directly into JavaScript, which is extremely helpful for full stack web development. Because of this feature, modern web-browsers don’t need to adopt Dart interpreters as most of them already interpret JavaScript.

Documentation, Manuals, and Tutorials

The best place to find documentation, interface guides, and libraries is at Dart's Guide Page, which includes links to several links to several aspects of the language. For those looking for a quick start to the language, Dart's Samples Page includes a great overview for syntax and programming examples, ranging from beginner content, like Hello World, to more complex topics, like interfaces and abstract classes. Dart's Resources Page also links users to books and videos aimed at all skill levels for the language. For those that prefer the help of fellow developers, Dart's Community Page includes several social media links to stay updated on releases, Reddit/StackOverflow links for more specific questions, and public repositories for those who would like to contribute to the Dart SDK or other Dart projects. Finally, DartPad has an in-browser demo that allows developers to run through real examples or develop and run their own programs.

Translators and installation instructions

Installation for Dart is fairly simple. Follow the instructions to install the Dart SDK to your system:

Windows

Go to https://gekorm.com/dart-windows/ and follow the instructions of the install wizard.

Alternatively, install Chocolatey, and run the following command:

$ -> choco install dark-sdk

macOS

Install Homebrew and run the following commands

$ -> brew tap dart-lang/dart
$ -> brew install dart

Linux (Ubuntu)

Perform this one-time setup

$ -> sudo apt-get update
$ -> sudo apt-get install apt-transport-https
$ -> sudo sh -c ‘wget -q0- https://dl=ssl.google.com/linux/linux_signing_key.pub | apt-key add -’
$ -> sudo sh -c ‘wget -q0- https://storage.googleapis.com/download.dartlang.org/linux /debian/dart_stable.list > /etc/apt/sources.list.d/dart_stable.list’

Then, install Dart with the following commands

$ -> sudo apt-get update
$ -> sudo apt-get install dart

More detailed instructions can be found on Dart.dev/get-dart


Standalone SDK (all platforms)

Standalone versions of the Dart SDK for Windows, Mac, and Linux can be downloaded in .zip format from the archive page. After extracting the SDK from the downloaded archive, it can be used to compile and run Dart code.

As an example, on Windows, a file can be executed from the command line by navigating to the directory containing the Dart executable (default:\dart-sdk\bin) and entering

dart file

where "file" is the path of a file containing Dart code.
